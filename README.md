# strapkit-lib
Contains npm modules used primarily by [StrapKit](https://github.com/strap/strapkit/).

## Setup
* Clone this repository onto your local machine. 
    `git clone https://github.com/strap/strapkit_lib.git`
* In terminal, navigate to the strapkit_lib directory.
    `cd strapkit_lib/`
* Install dependencies and npm-link
    `npm install && npm link`
* Navigate to StrapKit and link strapkit-lib
    `cd ../../strapkit && npm link strapkit-lib && npm install`
